# 🧪 QA - Fundamentos do Teste de Software

> Repositório criado durante minha transição de carreira para a área de **Quality Assurance (QA)**.

Este projeto reúne minhas anotações, estudos e exercícios desenvolvidos durante o curso **Fundamentos do Teste de Software**, servindo como base para a construção do meu portfólio profissional em QA.

---

# 📖 Objetivo

Compreender os principais conceitos de Teste de Software e Garantia da Qualidade, desenvolvendo uma base sólida antes de avançar para testes manuais, APIs e automação.

Ao final deste estudo, o objetivo é ser capaz de:

- compreender o papel do QA dentro do ciclo de desenvolvimento;
- identificar diferentes tipos de testes;
- elaborar casos de teste;
- documentar bugs de maneira profissional;
- entender o ciclo de vida de um defeito;
- conhecer boas práticas utilizadas pelo mercado.

---
